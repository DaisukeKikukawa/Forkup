 # 学習目標

## スキル
 - 3ヶ月後
   - CRUD等の基本機能の作成の動くものが作れるというレベルから、セキュリティ面やパフォーマンスを考慮した実装ができるようになる
 - 半年後
   - 既存のコードの修正だけでなく、新規機能の開発やより複雑な実装をできるようになる
 - 1年後
   - コードを書くだけでなく、案件定義からテストまで上流から下流まで一人でこなせるようになる
